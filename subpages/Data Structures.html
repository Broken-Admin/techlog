Data Structures

New ways to store information.

Data structures define the way primitive data is stored, such as numbers or character. There are many different types of data structures.

A commonly used data structure is an array, which consists of a primitive data types which are stored consecutively in memory.

Typically using an index with an array allows one to access specifc items with a O(1) time complexity..

A popular use for arrays are null-terminated strings. Which consist characters "strung" together in an array, terminated by a null character.

Another different but similar data structure is a linked list. A linked list has values which are not stored end-to-end in memory, but which contain the location of the next - and sometimes previous - node in the list alongside their own value or values.

In a linked list, the nth node can be accessed with an O(n) time complexity.

Although lists are common ways to store data, there are also many other ways in which it can be stored.

Stacks are an older but still used data structure. They consist of a top and bottom, in which the top grows away from the bottom as data is stored.

A stack almost always has 2 operations which can be performed on it, push and pop.

The push operation puts an item on the "top" of the stack, moving the top away from the bottom. A pop operation removes an item from the "top" of the stack and returns it in one form or another.

There are also other operations which are sometimes allowed, such as peeking at the top of the stack, which tells the "top" value on the stack without removing it.
