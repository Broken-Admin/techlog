Number Systems

Different ways to write your values.

Return to Catalog

You’ve likely used Roman numerals - I, II, III… - before and would be aware they’re a different style of writing numbers. Arabic numerals - 1, 2, 3... - are what you’re likely used to using. But you may be interested to know about the different ways to write values.

It is estimated that 20,000 years ago, during the time of the Upper Paleolithic period early quatanization began with tally marks. These marks later evolved into symbols which represent larger quantities as Roman numerals. Arabic numerals and positional notation were later developed.

Positional notation is the use of powers of the radix per each place moved, the earliest of these systems could be considered to be the Myan numeral system. You have likely been ingrained with the knowledge of the ones, tens, and hundreds places. These are all powers of the radix 10 in the base-10 system decimal, 100, 101, and 102. This radix can be changed to allow for further base systems.

Such as base-6 which uses 60, 61, and 62. This could allow for 10 in base-6 to mean 12 in base-10. For simplification of understanding I will begin denoting their base with a subscript. Such as 12510 denoting 125 in base-10.

A side note is that base-6 can be used to count on one’s hands, using your left hand as the 6ths place and your right hand as the 1s place. Allow for positional notation.

Each number system of base-n will have n symbols to represent all values. Therefore base-6 would only need symbols 0, 1, 2, 3, 4, 5.

In binary, base-2 only 2 digits can be used, base-9 only 9, base-11 has 11, and so on. This means that as the radix increases, the number of places necessary to represent a value decreases, see Radix economy for in-depth mathematics upon this. For example, although 24510 can be represented with 111101012 in binary’s base-two, only two places - F516 - are necessary to represent the value in hexadecimal’s base-16.

A common use of binary is digital computer systems. Where a wire can represent a single position via an on or or off signal, a 0 or a 1. Other base systems which utilize the powers of 2 are commonly used to ease usage instead of having to read direct binary or spend time converting to decimal’s base-10.
